# threejs-boilerplate

## command

```
# development (include start server)
yarn start

# build (publish files to public directory)
yarn build

# add package
yarn add <package>
yarn add -D <package>

# remove package
yarn remove <package>

# eslint
yarn lint
yarn lint:fix
```
